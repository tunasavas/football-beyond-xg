# Football Beyond xG

## Project Goal
This project analyzes whether expected goals (xG) is a reliable predictor of actual match outcomes.

## Dataset
- World Cup 2022 data
- Includes xG, goals, and performance metrics

## Methods
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Hypothesis testing (Pearson, Spearman, Mann-Whitney)

## Key Question
Do teams that outperform xG have better finishing ability or just luck?

## Files
- data.csv → dataset
- xg_analysis.ipynb → analysis
- project_proposal.pdf → proposal
